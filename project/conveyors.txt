
База данных conveyor предназначена для поддержки разработки конвейеров для горнообогатительных фабрик. 

База данных позволяет вести учет разработанной документации по всем компонентам конвейеров. Она позволяет решать такие задачи:
 - Учет хода разработки и разработанной документации
 - Получение номенклатуры разработанных конвейеров
 - Подбор аналогов узлов для будущих разработок
 - Анализ основных характеристик разработанных конвейеров 

База данных содержит реальные данные из моих прошлых проектов (импортированные из других форматов). 
Я планирую расширять и использовать ее в моей нынешней работе.
